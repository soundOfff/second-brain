# Obsidian

A markdown-based note vault application, cited as the **primary note tool** for storing
an [[concepts/llm-wiki]] [2026-06-23-karpathy-llm-wiki]. It is well suited to the method
because it stores plain markdown with frontmatter and supports `[[…]]` wiki-link
cross-references — the same format the agent writes. (The method does not require
Obsidian; any markdown vault works.)
