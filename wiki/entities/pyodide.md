# Pyodide

A distribution of Python that runs in the browser via **WebAssembly**

## In these sources

It is the runtime behind **Datasette Lite** — [[entities/datasette]] running entirely in
the browser. [[entities/simon-willison]] is investigating whether that in-browser stack
could edit **persistent SQLite files** on the user's computer using OPFS (Origin Private
